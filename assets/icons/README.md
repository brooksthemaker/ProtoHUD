# HUD icons — drop your art here

This folder is the icon library for ProtoHUD. The HUD loads `<name>.png` lazily
and **never recolors** it — author each one already-colored (white reads well for
a plain glyph). Missing names silently fall back to built-in vector glyphs, so the
HUD always shows *something*; the PNG just takes over when present.

**To use your own art:** overwrite any file below with a same-named PNG and rebuild
(`assets/icons/` is re-synced to the build output on every build). The included
PNGs are placeholders generated by `_make_filler_icons.py` — replace freely.

## Authoring notes
- Square PNG, RGBA with a **transparent background**. Any size (96–256 px is plenty);
  the loader fits it into its slot with aspect preserved and only scales alpha.
- Weather + notification icons render directly over the camera feed — give them a
  dark edge / outline so they stay legible on bright backgrounds.
- Status icons sit on a dark badge that the HUD draws, so a plain **white** glyph is
  ideal (the HUD dims it via alpha when the subsystem is inactive).

## Slots

### Weather (info-panel weather widget; mapped from WMO code in `wmo_icon()`)
| File | Condition |
|------|-----------|
| `wx-clear.png`        | clear, day |
| `wx-clear-night.png`  | clear, night |
| `wx-partly.png`       | partly cloudy, day |
| `wx-partly-night.png` | partly cloudy, night |
| `wx-cloudy.png`       | overcast |
| `wx-fog.png`          | fog |
| `wx-drizzle.png`      | drizzle |
| `wx-rain.png`         | rain / showers |
| `wx-snow.png`         | snow |
| `wx-storm.png`        | thunderstorm |

### Notifications (toasts + ALERTS widget; from `notif_type_icon()`, or per-notification override)
| File | Type |
|------|------|
| `alarm.png`   | alarm fired |
| `timer.png`   | timer elapsed |
| `message.png` | LoRa / incoming message |
| `bell.png`    | generic app notification |

### Status ring (badges around the info-panel disc; lit when active, dimmed when not)
| File | Subsystem |
|------|-----------|
| `status-wifi.png`    | Wi-Fi associated |
| `status-bt.png`      | Bluetooth device connected |
| `status-gamepad.png` | gamepad connected |
| `status-audio.png`   | spatial audio running |
| `status-ssh.png`     | SSH session active |
| `status-lora.png`    | LoRa radio up |

## Regenerating the placeholders
```
python3 assets/icons/_make_filler_icons.py
```
Pure stdlib (no Pillow/ImageMagick needed). Edit the per-icon functions there if you
want to tweak the fallback art.
