Background images for the ProtoHUD startup landing page.

Drop .png / .jpg / .jpeg / .bmp files here to ship them as defaults. Users can
also add their own to:  ~/protohud/backgrounds/

The landing page scans this folder first (bundled defaults), then the user
folder; entries are de-duplicated by file name and sorted. Pick one from the
landing page / Settings. Any resolution works (it's scaled to fill the screen);
~1920x1080 or wider looks best on the glasses.
